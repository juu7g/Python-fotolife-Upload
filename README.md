# Python-fotolife-Upload
はてなフォトライフへ画像アップロード

## 概要 Description
はてなフォトライフに画像をアップロードするアプリ   


## 特徴 Features

- 画像を exe にドラッグアンドドロップするだけでアップロードします。  
- はてなフォトライフのフォルダを指定できます。  
- ファイル名をタイトルに設定します。  
- はてな記法（fotolife記法）をログに出力します。   


## 依存関係 Requirement

- Python 3.8.5  
- Requests 2.25.1  

## 使い方 Usage

```dosbatch
	fotolifeUpload.exe
```
またはfotolifeUpload.exeのアイコンにアップロードしたいファイルをドラッグ＆ドロップします

- 操作 Operation  
	- アプリの起動  
	- 画像の選択  
		- ファイル選択ダイアログで画像を選択  
		- 起動時に画像をドラッグアンドドロップして画像を選択  
	- はてなフォトライフのアップロード先フォルダを指定  

## インストール方法 Installation

- pip install requests
